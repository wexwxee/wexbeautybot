# 💅 Бот для записи на маникюр

Полностью готовый Telegram-бот на Python с aiogram 3, SQLite и мини-аппом.

---

## 📁 Структура проекта

```
manicure_bot/
├── bot.py                    # Точка входа — запуск бота
├── config.py                 # ⚙️ НАСТРОЙКИ (заполнить перед запуском!)
├── requirements.txt          # Зависимости
│
├── database/
│   ├── __init__.py
│   └── db.py                 # Работа с SQLite
│
├── handlers/
│   ├── __init__.py
│   ├── user.py               # Обработчики для клиентов
│   └── admin.py              # Обработчики для администратора
│
├── keyboards/
│   ├── __init__.py
│   └── keyboards.py          # Все inline-клавиатуры
│
├── middlewares/
│   ├── __init__.py
│   └── subscription.py       # Проверка подписки на канал
│
├── utils/
│   ├── __init__.py
│   ├── states.py             # FSM состояния
│   └── scheduler.py          # APScheduler — напоминания
│
└── mini_app/
    └── index.html            # 🌐 Telegram Mini App (веб-интерфейс)
```

## ⚙️ Что нужно заполнить в config.py

```python
BOT_TOKEN = "7123456789:AAF..."          # Токен от @BotFather
ADMIN_ID = 123456789                      # Ваш Telegram ID (от @userinfobot)
CHANNEL_ID = -1001234567890               # ID канала (для проверки подписки)
CHANNEL_LINK = "https://t.me/mychannel"  # Ссылка на канал
SCHEDULE_CHANNEL_ID = -1001234567890      # Канал для публикации расписания
```

### Как получить данные:
- **BOT_TOKEN** — создайте бота через @BotFather → /newbot
- **ADMIN_ID** — напишите @userinfobot, он пришлёт ваш ID
- **CHANNEL_ID** — добавьте @username_to_id_bot в канал и он пришлёт ID
- **CHANNEL_LINK** — ссылка вида https://t.me/ваш_канал

## 🚀 Установка и запуск

### 1. Клонируйте/скачайте проект
```bash
cd manicure_bot
```

### 2. Создайте виртуальное окружение (рекомендуется)
```bash
python3 -m venv venv
source venv/bin/activate       # Linux/Mac
# или
venv\Scripts\activate          # Windows
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Заполните config.py
Откройте `config.py` и замените все значения на свои.

### 5. Запустите бота
```bash
python bot.py
```

## 🌐 Подключение Mini App

1. Загрузите папку `mini_app/` на любой хостинг с HTTPS:
   - [Vercel](https://vercel.com) — бесплатно
   - [Netlify](https://netlify.com) — бесплатно
   - GitHub Pages — бесплатно

2. В @BotFather выполните: `/newapp` → укажите URL вашего хостинга

3. Добавьте кнопку в меню бота через @BotFather:
   ```
   /setmenubutton → выберите бота → Web App → укажите URL и название
   ```

## 📋 Функционал

### Для клиентов:
- ✅ Просмотр доступных дней через инлайн-календарь
- ✅ Выбор свободного временного слота
- ✅ Ввод имени и телефона
- ✅ Подтверждение записи
- ✅ Отмена своей записи
- ✅ Просмотр прайса
- ✅ Ссылка на портфолио
- ✅ Напоминание за 24 часа до визита
- ✅ Обязательная проверка подписки на канал

### Для администратора (`/admin`):
- ✅ Добавление рабочих дней (вручную или авто на 30 дней)
- ✅ Добавление/удаление временных слотов
- ✅ Открытие/закрытие рабочих дней
- ✅ Просмотр расписания на дату
- ✅ Отмена записей клиентов с уведомлением

---

## 🔔 Напоминания

- Используется APScheduler
- Напоминание отправляется ровно за 24 часа до визита
- Если запись создана менее чем за 24ч — напоминание не создаётся
- При отмене записи напоминание автоматически удаляется
- При перезапуске бота все напоминания восстанавливаются из БД

---

## ⚠️ Важные замечания

1. **CHANNEL_ID** — бот должен быть **администратором** канала для проверки подписок
2. **SCHEDULE_CHANNEL_ID** — бот должен быть **администратором** канала для публикации
3. База данных `manicure.db` создаётся автоматически при первом запуске
4. Для production-сервера рекомендуется использовать `systemd` или `supervisor` для автозапуска
